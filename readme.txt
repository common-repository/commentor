A wordpress comment plugin

== Description ==

Commentor is a wordpress plugin to help you customize wordpress default comments template.
Also commentor add comments like feature for your wordpress site.

= Personalization settings =

1. Form direction
2. Colors (Change all colors related to your theme color)
3. Form styling
4. Display

== Installation ==

1. Upload the entire `/commentor` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
